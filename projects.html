Project Name : WeText

Technologies Used : React,Express,MongoDB

Project Description :


  • Implemented a user friendly interface using react.

  • Server is created using express framework.

  • Integrated mongodb database to store user details.

  • Integrated an avatar api which will generate a random avatar based on gender selection from registration page. If user selects male a male random avatar will be generated. if female then a female random avatar will be generated.

  • after user registers or login a cookie will be generated and it will not be expired for 15 days until user clicks on logout button. If user closes the browser without clicking on logout button cookie will not be expired. so if user again visits the WeText website they can easily login without entering their credentials.
  • * Working on socket io to enable real time communication. *
